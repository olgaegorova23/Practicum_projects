В рамках проекта я проанализировала работу сайта онлайн-магазина строительных материалов и товаров для дома. 
Заказчик столкнулся с рядом проблем: 
- частое закрытие сайта на этапе оплату заказа;
- маленькое количество заказов с онлайн-оплатой;
- частые отказы по факту доставки;
- отсутствие синхронизации данных между сайтом магазина и складскими остатками.

Моей задачей было восстановить весь процесс работы онлайн-магазина в процессе оформления заказа на сайте и последующей доставки, а также определить точки роста и необходимые доработки.

После проведенного анализа я спроектировала BPMN диаграмму TO_BE процессов, а так же диаграммы классов и объектов.
Все диаграммы вошли в итоговые отчеты по обследованию и ТЗ, которые были представлены заказчику.

Для работы использовала сервис draw.io
